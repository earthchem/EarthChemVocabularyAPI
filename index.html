EarthChem Vocabulary Services
The EarthChem Vocabulary Services are a set of RESTful web services intended to simplify the process of maintaining controlled vocabularies within the IEDA community. All data currently comes from a dev instance of the EarthChem ODM2 database and is JSKOS encoded to easily implement within client-side applications.

Below are details for each of the services currently enabled:
Chemical Analysis Type Service:
Endpoint URI:
http://vocab.earthchemportal.org/vocabulary/chemicalAnalysisType
Example Detail URI:
http://vocab.earthchemportal.org/vocabulary/chemicalAnalysisType/catype20
Example Query:
http://vocab.earthchemportal.org/vocabulary/chemicalAnalysisType?label=a
